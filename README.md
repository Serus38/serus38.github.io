# Mi Portafolio Web

Estructura base para un portafolio web profesional.

## 📁 Estructura del Proyecto

```
MyProfile/
├── index.html           # Página principal
├── css/
│   └── style.css       # Estilos principales
├── js/
│   └── script.js       # Funcionalidades JavaScript
├── assets/
│   └── images/         # Carpeta para imágenes
└── README.md           # Este archivo
```

## 🚀 Características

- **Responsivo**: Diseño adaptable a todos los dispositivos (mobile, tablet, desktop)
- **Navegación moderna**: Menú con efecto hamburguesa en móviles
- **Secciones incluidas**:
  - Hero (Bienvenida)
  - Acerca de mí
  - Portafolio de proyectos
  - Formulario de contacto
  - Footer con redes sociales
- **Animaciones suaves**: Transiciones y efectos visuales
- **Paleta de colores moderna**: Gradientes atractivos

## 📝 Cómo usar

1. **Personaliza el contenido**:
   - Reemplaza "[Tu Nombre]" en index.html con tu nombre
   - Añade tus proyectos en la sección de proyectos
   - Actualiza la información en la sección "Acerca de mí"

2. **Añade imágenes**:
   - Coloca tus imágenes en la carpeta `assets/images/`
   - Actualiza las rutas en index.html

3. **Personaliza estilos**:
   - Modifica los colores en las variables CSS en `css/style.css`
   - Ajusta la tipografía según tus preferencias

4. **Abre el sitio**:
   - Abre `index.html` en tu navegador
   - O usa un servidor local (ej: Live Server)
